# Installation

1. Download `rogue-white-label-crm.zip` from GitHub Releases.
2. In WordPress, go to **Plugins → Add New → Upload Plugin**.
3. Upload the ZIP file.
4. Activate **Rogue White Label CRM**.
5. Go to **Rogue CRM → Settings** and enter your business details.
6. Add customers and jobs from the CRM menu.

## Recommended first setup

- Set your business name and email.
- Set your job prefix, for example `JOB-`, `CASE-`, or `WORK-`.
- Replace the default statuses with your own workflow.
- Create CRM users through **Users → Add New**.
- Assign trusted users the **CRM Manager** role.
- Assign basic staff the **CRM Staff** role.
