# GitHub Release Notes Template

## Release title

Rogue White Label CRM v1.0.0

## Tag

`v1.0.0`

## Description

Initial release of Rogue White Label CRM, a reusable white-label CRM plugin for WordPress.

### Included

- Customers
- Jobs
- Job delete function
- Customer delete function
- Configurable job statuses
- Configurable job prefix
- White-label business settings
- CRM Manager and CRM Staff roles
- CSV exports
- GitHub README and documentation
- Release graphic and logo assets

### Installation

Download `rogue-white-label-crm.zip`, then upload it through **WordPress Admin → Plugins → Add New → Upload Plugin**.
