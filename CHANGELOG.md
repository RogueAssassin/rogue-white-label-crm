# Changelog

## 1.0.0 - Initial Release

- Initial white-label CRM plugin release.
- Added customer management.
- Added job management with configurable job prefix and statuses.
- Added delete buttons for jobs and customers.
- Added CRM dashboard cards.
- Added white-label settings for business details, colours, currency, job statuses, and PDF terms.
- Added CSV export for customers and jobs.
- Added CRM Manager and CRM Staff roles.
- Added GitHub-ready README, docs, assets, and release graphic.
